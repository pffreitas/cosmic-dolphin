## 2024-04-30 - Added keyboard shortcut hints to interactive elements
**Learning:** Adding visual keyboard shortcut hints (like `<kbd>⌘ K</kbd>`) directly to interactive elements (like buttons) significantly improves their discoverability and teaches users power-user workflows.
**Action:** Always wrap modifier keys and shortcuts in semantic `<kbd>` elements to provide accessible markup and visual distinctiveness for keyboard shortcut hints in UI components.
